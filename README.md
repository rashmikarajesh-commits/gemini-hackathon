# 📚 Gen-Z Study Slay
**Turning mid academic text into iconic Slay-Maries.**

## ✨ Inspiration
Academic textbooks are often "mid"—dry, long, and hard to focus on. We wanted to give students "main character energy" by translating boring text into something actually fun to read.

## 🚀 What it does
Gen-Z Study Slay takes complex academic paragraphs and uses **Gemini 3 Flash Preview** to generate a "Slay-Mary." It breaks down core concepts into 3 iconic bullet points filled with Gen-Z slang and emojis.

## 🛠️ How to Run
1. Clone the repo: `git clone https://github.com/rashmikarajesh-commits/gemini-hackathon.git`
2. Install requirements: `pip install google-generativeai`
3. Add your Gemini API Key in `main.py` where it says `PASTE_YOUR_API_KEY_HERE`.
4. Run the script: `python main.py`
